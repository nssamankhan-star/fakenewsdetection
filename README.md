 📰 Fake News Detection

 📌 Overview

This project uses **Natural Language Processing (NLP)** and **Machine Learning** to classify news articles as **Real** or **Fake**.

## 🚀 Features

* Text preprocessing using **NLTK** and **spaCy**
* Word cloud visualization for text analysis using **PIL (Pillow)** and **WordCloud**
* Feature extraction using **TF-IDF**
* Machine learning model for fake news classification

## 🛠️ Technologies Used

* Python
* Pandas
* NLTK
* spaCy
* Scikit-learn
* TF-IDF
* WordCloud
* Pillow (PIL)

## ▶️ How to Run

1. Clone the repository.
2. Install the required libraries:

   ```bash
   pip install -r requirements.txt
   ```
3. Run the project:

   ```bash
   python fake_news_detection.py
   ```

## 📈 Workflow

1. Clean and preprocess the text.
2. Visualize the data with word clouds.
3. Extract features using TF-IDF.
4. Train the machine learning model.
5. Predict whether news is **Real** or **Fake**.
